## 💻 Local Setup & Development

To run this project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Mayank2014l/Portfolio.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd Portfolio
   ```
3. **Run it:**
   Simply open `index.html` in your favorite web browser or start a local dev server (e.g., Live Server in VS Code).
